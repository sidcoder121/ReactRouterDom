## 📈 Key Takeaways & Notes

### 💡 Core Pillars of React
1. **Component-Based Architecture**: Building reusable, isolated pieces of UI.
2. **Declarative UI**: Telling React *what* the UI should look like based on the current state, instead of manually manipulating the DOM.
3. **Unidirectional Data Flow**: Data flows downward from parent to child components via props, making debugging predictable.

> 📝 **Pro Tip Learned:** Never mutate state directly! Always use the setter function provided by `useState` to trigger a re-render.
